Type definitions for parsing and handling VAST (Video Ad Serving Template) 4.x responses.

<Note>
  These types conform to the IAB VAST 4.x specification. For more information, see the [official VAST documentation](https://iabtechlab.com/standards/vast/).
</Note>

## VASTResponse

Root VAST response structure returned by the parser.

```typescript
interface VASTResponse {
  version: string;
  ads: Ad[];
  errors: string[];
}
```

<ResponseField name="version" type="string">
  VAST version from the XML (e.g., "4.0", "4.1", "4.2").
</ResponseField>

<ResponseField name="ads" type="Ad[]">
  Array of parsed [Ad](#ad) elements from the VAST response.
</ResponseField>

<ResponseField name="errors" type="string[]">
  Error tracking URLs from the VAST document.
</ResponseField>

---

## Ad

Represents a single ad element. Can be either inline or wrapper.

```typescript
interface Ad {
  id: string;
  sequence?: number;
  adSystem?: AdSystem;
  adTitle?: string;
  impressions: Impression[];
  creatives: Creative[];
  errors: string[];
  wrapper?: Wrapper;
  inLine?: InLine;
}
```

<ResponseField name="id" type="string">
  Unique identifier for the ad.
</ResponseField>

<ResponseField name="sequence" type="number">
  Playback sequence for ad pods. Lower numbers play first.
</ResponseField>

<ResponseField name="adSystem" type="AdSystem">
  Information about the ad server. See [AdSystem](#adsystem).
</ResponseField>

<ResponseField name="adTitle" type="string">
  Title of the advertisement.
</ResponseField>

<ResponseField name="impressions" type="Impression[]">
  Impression tracking URLs. See [Impression](#impression).
</ResponseField>

<ResponseField name="creatives" type="Creative[]">
  Creative elements (linear, non-linear, companion). See [Creative](#creative).
</ResponseField>

<ResponseField name="errors" type="string[]">
  Error tracking URLs specific to this ad.
</ResponseField>

<ResponseField name="wrapper" type="Wrapper">
  Present if this is a wrapper ad that points to another VAST tag. See [Wrapper](#wrapper).
</ResponseField>

<ResponseField name="inLine" type="InLine">
  Present if this is an inline ad with direct creative content. See [InLine](#inline).
</ResponseField>

  maxBitrate?: number;

<ResponseField name="type" type="string">
  MIME type with optional codec (e.g., `video/mp4`, `video/mp4; codecs="avc1.42E01E"`).

<ResponseField name="width" type="number">
  Video width in pixels.
</ResponseField>

<ResponseField name="height" type="number">
  Video height in pixels.
</ResponseField>

<ResponseField name="bitrate" type="number">
  Target bitrate in kbps.
</ResponseField>

<ResponseField name="minBitrate" type="number">
  Minimum bitrate for adaptive streaming.
</ResponseField>

<ResponseField name="maxBitrate" type="number">
  Maximum bitrate for adaptive streaming.
</ResponseField>

<ResponseField name="scalable" type="boolean">
  Whether the media file can be scaled.
</ResponseField>

<ResponseField name="maintainAspectRatio" type="boolean">
  Whether to maintain aspect ratio when scaling.
</ResponseField>

<ResponseField name="codec" type="string">
  Video codec identifier.
</ResponseField>

<ResponseField name="apiFramework" type="string">

## VASTErrorCode

Standard VAST error codes per IAB specification.

```typescript
enum VASTErrorCode {
  XML_PARSING_ERROR = 100,
  VAST_SCHEMA_VALIDATION_ERROR = 101,
  VAST_VERSION_NOT_SUPPORTED = 102,
  GENERAL_WRAPPER_ERROR = 300,
  WRAPPER_TIMEOUT = 301,
  WRAPPER_LIMIT_REACHED = 302,
  NO_VAST_RESPONSE = 303,
  GENERAL_LINEAR_ERROR = 400,
  FILE_NOT_FOUND = 401,
  MEDIA_TIMEOUT = 402,
  MEDIA_NOT_SUPPORTED = 403,
  GENERAL_COMPANION_ERROR = 600,
  UNDEFINED_ERROR = 900
}
```

<ResponseField name="XML_PARSING_ERROR" type="100">
  Failed to parse VAST XML.
</ResponseField>

<ResponseField name="VAST_SCHEMA_VALIDATION_ERROR" type="101">
  VAST XML does not conform to schema.
</ResponseField>

<ResponseField name="VAST_VERSION_NOT_SUPPORTED" type="102">
  VAST version is not supported by the player.
</ResponseField>

<ResponseField name="GENERAL_WRAPPER_ERROR" type="300">
  General wrapper ad error.
</ResponseField>

<ResponseField name="WRAPPER_TIMEOUT" type="301">
  Wrapper ad request timed out.
</ResponseField>

<ResponseField name="WRAPPER_LIMIT_REACHED" type="302">
  Maximum wrapper depth exceeded.
</ResponseField>

<ResponseField name="NO_VAST_RESPONSE" type="303">
  No VAST response received from wrapper.
</ResponseField>

<ResponseField name="GENERAL_LINEAR_ERROR" type="400">
  General linear creative error.
</ResponseField>

<ResponseField name="FILE_NOT_FOUND" type="401">
  Media file not found or failed to load.
</ResponseField>

<ResponseField name="MEDIA_TIMEOUT" type="402">
  Media file request timed out.
</ResponseField>

<ResponseField name="MEDIA_NOT_SUPPORTED" type="403">
  No supported media files available.
</ResponseField>

<ResponseField name="GENERAL_COMPANION_ERROR" type="600">
  General companion ad error.
</ResponseField>

<ResponseField name="UNDEFINED_ERROR" type="900">
  Undefined error occurred.
</ResponseField>
